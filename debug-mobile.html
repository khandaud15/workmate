Mobile Debug Test

Open this on your mobile device and test the job search at: https://talexus.ai/dashboard/jobs

Debug Steps:

  1. Open browser console on mobile (Chrome: Menu > More tools > Developer tools)
  2. Fill in job title: "data scientist"
  3. Fill in location: "US"
  4. Check if search button becomes enabled (loses gray color)
  5. Tap search button
  6. Watch console for any errors

Common Mobile Issues:

  • Touch events not working properly
  • Input focus/blur issues on mobile keyboards
  • Viewport scaling affecting click targets
  • Mobile browser differences (Safari vs Chrome)
  • Network timeouts on mobile connections
